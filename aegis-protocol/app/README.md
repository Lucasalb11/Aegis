# Aegis Protocol Frontend

A Next.js frontend for interacting with Aegis Protocol on Solana.

## Features

- Wallet connection (Phantom, Solflare)
- Pool creation and management
- Liquidity provision
- Token swapping

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Usage

1. Connect your Solana wallet
2. Create a new liquidity pool
3. Add liquidity to existing pools
4. Perform token swaps

## Technologies

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Solana Wallet Adapter
- Aegis SDK
